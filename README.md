# README

## Name

gqtp

## Description

Gqtp gem is a GQTP (Groonga Query Transfer Protocol) Ruby
implementation.

Gqtp gem provides both GQTP client and GQTP server implementations.
They provide asynchronous API. You can use gqtp gem for high
concurrency use.

## Install

```
% gem install gqtp
```
